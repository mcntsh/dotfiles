## Pre-requisites

1. Install [Homebrew](https://brew.sh/)

```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

2. Install [tmux](https://github.com/tmux/tmux)

```
brew update
brew install tmux
```

## Installation

```
cp .tmux.conf $HOME/.tmux.conf
```
